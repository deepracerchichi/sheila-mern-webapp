# Sheila - MERN Notes Web Application
---
A modern, full-stack note-taking web application built with the MERN stack (MongoDB, Express.js, React, Node.js). Sheila provides users with a clean, intuitive interface to create, manage, and organize their notes with a sleek dark theme and vibrant design.
- **[Live Demo](https://sheila-mern-webapp.onrender.com)**
##  Features
- **Full CRUD Operations** - Create, read, update, and delete notes seamlessly.
- **Modern UI/UX** - Built with DaisyUI for a polished, responsive interface.
- **Responsive Design** - Works beautifully on desktop, tablet, and mobile.
- **Real-time Updates** - Instant note creation and modifications.
- **Rate Limiting** - API protection using Upstash Redis.
- **Card-based Layout** - Elegant grid display with quick action buttons.
- **Timestamps** - Track when notes were created/modified.
- **Production Ready** - Deployed and optimized for performance.


## Tech Stack
### Frontend
- **React.js** - UI framework
- **DaisyUI** - Tailwind CSS component library
- **Tailwind CSS** - Utility-first styling
- **Axios** - HTTP requests

### Backend
- **Node.js** - Runtime environment
- **Express.js** - Web framework
- **MongoDB** - NoSQL database
- **Mongoose** - MongoDB object modeling
- **Upstash Redis** - Rate limiting

### Deployment
- **Render** - Cloud hosting platform
- **MongoDB Atlas** - Cloud database

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/yourusername/sheila-mern-webapp.git
cd sheila-mern-webapp
```

2. **Install backend dependencies**
```bash
cd backend
npm install
```

3. **Install frontend dependencies**
```bash
cd ../frontend
npm install
```

4. **Run the application**

Backend:
```bash
cd backend
npm start
```

Frontend (in a new terminal):
```bash
cd frontend
npm start
```
